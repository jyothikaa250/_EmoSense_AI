About EmoSense AI

EmoSense AI is an intelligent machine learning based web application designed to detect human emotions from speech signals. It enables computers to understand human feelings by analyzing voice input in real-time.

The system allows users to upload audio or record voice directly through the browser. The input audio is processed using advanced audio signal processing techniques, and important features such as MFCC, spectral features, and energy levels are extracted for analysis.

To improve reliability, the system includes validation mechanisms such as silence detection and music filtering, ensuring that only valid human speech is used for emotion prediction.

Key Features

  • 🎤 Voice Recording for real-time emotion analysis
  • 📁 Audio Upload for emotion prediction
  • 🧠 Machine Learning based emotion classification (SVM)
  • 🎵 Music Detection to filter non-speech input
  • 🔇 Silence Detection for invalid audio handling
  • 📊 Top-3 Emotion Prediction with confidence score
  • ⭐ User Feedback System for model evaluation
  • 📈 Admin Dashboard for monitoring predictions

Technology Stack

Python, Django, Machine Learning (Scikit-learn), Librosa for audio processing, and HTML, CSS, JavaScript for frontend development.

System Workflow

Audio Input → Preprocessing → Validation (Silence & Music Detection) → Feature Extraction → Emotion Prediction → Result Display & Storage → Feedback Collection
